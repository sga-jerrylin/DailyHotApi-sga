```yaml

  backend:   # Node.js API服务
  frontend:  # Next.js + Nginx 静态服务  
  redis:     # Redis 缓存服务
```

### 网络配置：
- ✅ **Docker网络** - dailyhot-network 内部通信
- ✅ **端口映射** - 3000(前端), 6688(后端), 6379(Redis)
- ✅ **API代理** - Nginx反向代理到后端服务

### 数据流程：
```
前端 → Nginx → 后端API → Redis缓存 → 外部数据源
     ↓
DeepSeek AI API (直接调用)
```
